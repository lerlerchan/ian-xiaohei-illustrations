# Chibi Character IP

## 角色定义

Chibi 是本配图系统的固定视觉 IP，是用户自己的 cute chibi 形象。

默认每张图都要出现 chibi。Chibi 不是贴纸，不是可爱装饰，而是正在认真参与系统运转的可爱工作者。Chibi 必须承担画面的核心动作，不能只是站在旁边当装饰。

## 外形

**比例**
- 大头小身体：头部约占总高度的 35–45%。
- 身体短而圆润，四肢简单软萌。
- 整体轮廓清晰，剪影一眼可读。

**脸部**
- 大而简洁的眼睛（圆形或半圆形，有神但不夸张）。
- 细腻但简短的鼻子和嘴（情绪由眉眼传达，不需要复杂嘴型）。
- 圆润的脸颊，柔软的下巴。

**发型**
- 发型轮廓干净，线条简洁。
- 可有短发、马尾、刘海等，保持与角色整体比例协调。

**手脚**
- 手部圆润简化，不画复杂手指。
- 脚/鞋子扁平圆润，轮廓简单。

**线条**
- 全黑线稿，轻微抖动感，非机械矢量。
- 线条均匀，无厚重轮廓。
- 极少阴影，仅在必要时用细线暗示立体。

## 服装

每张图可以根据场景选择服装：

| 场景 | 建议服装 |
|------|----------|
| 编码 / 专注模式 | 宽松卫衣 + 短裤 |
| 熬夜 / 3am energy | 睡衣 / 宽大上衣 |
| 学习 / 阅读 | 毛衣或 cardigan |
| 咖啡 / 休闲模式 | 简单 T 恤 + 随意发型 |
| 通用 / 默认 | 宽松帽 T（oversized hoodie）|

服装是身份标签，不是重点；保持剪影简单，不要添加复杂印花或细节。

## 性格

- 可爱但认真。
- 有点困、有点专注、有点自得其乐。
- 不是夸张卖萌，而是日常做事的真实状态。
- 用于科技 / 工作场景，情绪是 cozy + focused，不是兴奋或搞怪。

## 动作库

让 chibi 承担核心动作：

- 坐在地上 / 靠墙，专注地用笔记本电脑。
- 抱着书或文件夹，皱眉思考。
- 拿着咖啡盯着屏幕发呆。
- 趴在桌上，一边打字一边打哈欠。
- 站在某个怪机器/系统旁，认真操作一个简单动作。
- 推、拉、搬运、指向、记录、挂标签、打开门、把东西塞进某个装置。
- 卡在某个结构里，但仍然冷静地工作。

动作目标：让 chibi 成为让画面成立的原因，而不是可有可无的点缀。

## 与场景物件的关系

Chibi 应该和场景物件产生实际互动：

- 笔记本电脑、咖啡杯、便利贴、书本、线缆 — 日常 tech-life 道具。
- 怪机器、漏斗、框、门、路径 — 系统结构隐喻道具。
- 手写注释标签 — 可以飘在 chibi 旁边，像思绪或说明。

## 禁止

- 不要让 chibi 只是站在角落里。
- 不要让 chibi 表情夸张、过度搞怪或歇斯底里。
- 不要画成过度商业化的游戏角色或品牌吉祥物。
- 不要加渐变、光晕、阴影、彩色填充。
- 不要加复杂服装图案、纹理或配件。
- 不要让 chibi 变成和构图无关的装饰贴纸。

## 判断标准

如果去掉 chibi，图的核心隐喻还能完全成立，说明 chibi 太装饰了；要重写提示词，让 chibi 成为动作主体。
